```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset='utf8' />
        <title>My Awesome Website</title>
        <style>
            /* Selects only h1 in the document */
            h1 {
                background: yellow;
            }
        </style>
    </head>
    <body>
        <h1>My Awesome Website</h1>
    </body>
</html>
```

The problem with this way is that it can only be used to provide styles for only one HTML document.
