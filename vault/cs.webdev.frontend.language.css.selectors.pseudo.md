When a link has been actually visited:

```css
a:visited {
    propertyName: propertyValue;
}
```

When the mouse hovers over a link:

```css
a:hover { 
    propertyName: propertyValue; 
}
```

When the link is actually being clicked on:

```css
a:active { 
    propertyName: propertyValue; 
}
```
